# Beta README

## Build

- Build: local beta candidate from current commit.
- Supported target: iPhone portrait, iOS 17+ simulator/device build.
- Primary verification device: SE-class simulator plus standard iPhone simulator.

## Run Structure

- A full run has 30 stages across six acts.
- Boss tables are stages 5, 10, 15, 20, 25, and 30.
- Stage 30 is the only route to Casino Cleared.
- Ordinary clear cadence: stage result, reward draft, shop, next stage.
- Boss clear cadence: boss reward, shop, next stage.

## Tester Instructions

- Start from a clean install for first-pass testing.
- Record run seed, stage, hand, bankroll, Chips, Heat, selected Contact, and any visible issue.
- In debug/beta builds, use the QA menu to start at acts, boss stages, or resource fixtures.
- Include diagnostics export text with bug reports.

## Privacy

Diagnostics are intended to include gameplay state only: build context, seed, stage, hand, resources, selected Contact, active modifiers, and recent game-domain events. No personal data is intentionally collected.
