# Balance Report

## Pacing Target

- Ordinary stage: 2-4 minutes for first-time testers.
- Boss stage: 4-6 minutes for first-time testers.
- First-time full run: 90-150 minutes.
- Experienced full run: 60-100 minutes.

## Current Tuning Notes

- Stage count is 30 with deterministic boss stages.
- Wager bands scale by act without adding a hidden bankroll-percentage cap.
- Stage objectives currently emphasize solvency through fixed hand counts.
- Reward/shop cadence is consistent and data-driven enough for beta feedback.

## Simulation Status

- Existing deterministic tests verify stage topology and full Stage 30 transition through debug clear.
- A 10,000-run simulation gate remains a beta exit task and is listed in known issues rather than overstated as complete.
